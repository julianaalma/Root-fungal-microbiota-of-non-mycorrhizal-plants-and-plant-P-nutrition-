# Root-fungal-microbiota-of-non-mycorrhizal-plants-and-plant-P-nutrition-
This repository contains all the coded needed to reproduce the analyses performed by Bruyant et al., 2025.
Authors: Pauline Bruyant1,2, Lauren Gillespie1, Jeanne Doré1, Pierre Emmanuel Courty2, Yvan Moënne-Loccoz1,3, Juliana Almario1

Each folder contains the scripts needed to reprodice the figures. Librarues need to be installed and the workid directory has to be adjusted.
The file "Mothur_Amplicon_seq_data_processing.txt" summarizes data processing in Mothur. 

All sequencing data are deposited at the NCBI database under accession PRJNA1141433 (Illumina Miseq)
